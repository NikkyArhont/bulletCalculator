# Ballistic Calculator (Bullet Calculator)

Мобильное приложение для баллистических расчетов, построенное на Flutter с интеграцией Firebase.

## 🚀 Текущий статус проекта

Проект находится в стадии активной разработки UI и интеграции с данными. Основные модули авторизации и управления профилем/арсеналом завершены.

### Реализованные функции:

1.  **Авторизация (Auth):**
    *   Вход по номеру телефона + SMS-код.
    *   Реализован Loader на кнопке подтверждения.
    *   Заблокирована возможность возврата назад во время процесса верификации (PopScope).
2.  **Профиль пользователя (Profile):**
    *   Просмотр данных пользователя и навигация к расчету/настройкам.
    *   **Настройки:** редактирование имени/фамилии и выбор единиц измерения (дистанция, температура, давление).
    *   Данные сохраняются атомарно в Firestore (`users/{uid}`).
    *   Кастомные диалоги для **Выхода** (зеленая тема) и **Удаления аккаунта** (красная тема).
3.  **Арсенал (Weapons Management):**
    *   **Экран добавления оружия (AddWeapon):** современный интерфейс для ввода параметров (высота прицела, твист, скорость, БК и т.д.).
    *   Интерактивное переключение направления нарезов.
    *   Сохранение в Firestore: `users/{uid}/weapons`.
4.  **Баллистический расчет (Shoot Page):**
    *   **Интерактивный селектор оружия:** Выпадающий список, который в реальном времени подгружает данные из Firestore.
    *   Анимированное раскрытие списка, подсветка выбранного оружия и кнопка быстрого перехода к добавлению нового оружия.

## 🛠 Технический стек

*   **Framework:** Flutter (архитектура экспортирована из FlutterFlow).
*   **Backend:** Firebase Auth, Cloud Firestore.
*   **State Management:** FlutterFlow Model (постраничные модели).
*   **Design:** Custom Vanilla CSS/Flutter Themes, Google Fonts (Inter, Space Grotesk).

## 📂 Структура данных (Firestore)

*   `users/{uid}`:
    *   `display_name`, `last_name`
    *   `distance_unit`, `temp_unit`, `pressure_unit`
*   `users/{uid}/weapons/{weaponId}`:
    *   `name`, `caliber`, `sight_height`, `twist`, `twist_direction`, `muzzle_velocity`, `bc_model`, `bc_value` и др.
